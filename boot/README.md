# Boot Application

A simple Java Spring Boot project set up by Kalesha Vali.

## Features
- Spring Boot setup
- REST API sample (`/` endpoint)
- Maven build
- Clean architecture structure (controller, service, model, etc.)

## Getting Started

### Prerequisites
- Java 17+
- Maven 3+

### Running the App
```bash
./mvnw spring-boot:run
```

Visit: `http://localhost:8080/`
